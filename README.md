# Virtual File System Architecture

## Project Description
A command-line simulation of a hierarchical file system. It creates a virtual environment where users can manage directories and navigate using shell-like commands.

It utilizes a generic **Tree** structure for the folders and a **Stack** for undo/redo operations.

## Technical Details
* **Language:** Java
* **Structures:** Generic Tree, Stack, Recursion.
* **Features:** Nested directory management, navigation history, and recursive traversal.

## How to Run
Compile and run the application:
```bash
javac Main.java
java Main
